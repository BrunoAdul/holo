# Figma to Code

## Overview
This project focuses on converting Figma designs into functional web pages using HTML, CSS, and JavaScript. It serves as a bridge between UI/UX design and front-end development, ensuring that pixel-perfect designs are transformed into responsive, interactive websites.

## Features
- Converts Figma designs into clean, structured HTML and CSS code.
- Uses JavaScript to add interactivity and animations.
- Ensures responsive design for different screen sizes.
- Optimized for performance and accessibility.

## Technologies Used
- **HTML5**: For semantic and structured markup.
- **CSS3**: For styling, including Flexbox and Grid for layout.
- **JavaScript (ES6+)**: For dynamic and interactive elements.
- **Figma**: Used as the design source.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/brunoadul/holo.git
   cd holo
   ```
2. Open `index.html` in a web browser.
3. Modify `style.css` for styling adjustments.
4. Use `script.js` to add JavaScript functionalities.
